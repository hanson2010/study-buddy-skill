# StudyBuddy 快速上手指南（Quick Start）

本文件为 StudyBuddy AI 学习助手的核心工作手册，供 Agent 与开发者快速查阅核心指令、存储结构和关键规则。

---

## 💾 数据存储目录速览

数据存储在本地环境变量 `STUDYBUDDY_DATA_DIR` 指定的目录中。必须设置该环境变量，否则会提醒用户设置后再继续使用。

```
<STUDYBUDDY_DATA_DIR>/
├── profile.md                      # 考生的基本背景档案（姓名、选科、教材版本、目标分数与院校等，纯 Markdown 表格，无 YAML）
├── _index.md                       # 全局学习总览（包含各科掌握度、历次考试成绩记录、全局跨学科薄弱点）
├── colleges/                       # 高校与目标对齐数据仓
│   ├── _index.md                   # 高校资料索引（目标院校专业组列表、外部参考资料与分析规划报告的索引）
│   ├── YYYY/                       # 按招生年度归档：外部高校招生简章、专业组选科要求、历年录取投档线与分数分布数据
│   └── YYYY/MM/                    # 按执行年月归档：学情对齐中产生的「差距分析报告」与「个性化学习规划书」
├── raw/                            # 原始图片、会话日志归档（按执行年月归档 YYYY/MM/）
├── reports/                        # 自动生成的周报/月报（按报告年月归档 reports/YYYY/MM/，含 YAML 元数据）
└── subjects/                       # 六科学科数据仓（按学科分类）
    └── <学科>/
        ├── _index.md               # 学科索引（该学科高频主题/考点、当前学习目标、薄弱点表与正确率趋势）
        └── YYYY/MM/                # 学科日常错题、随堂练习与辅导文件（文件名遵循 YYYY-MM-DD-<slug>.md）
```

## ⚠️ 核心行为规范（Agent 必须遵守）

### Rule 1. 开口前先检索学情
*   每次开始辅导、批改或讲解前，**必须先读取 Root `_index.md`、学科 `_index.md` 和 `profile.md`**。
*   如果当前讨论或练习涉及历史标记为 `⚠️ 待巩固` 的薄弱点，**必须在输出中高亮提示**并引导复习。

### Rule 2. 智能辅导六步法
所有辅导/批改流程必须按以下路径执行（部分步骤根据子命令可做相应跳过/适配）：
1. **诊断/识别**：OCR 识别题目判断对错，或定位学习诉求。
2. **错因/盲区分析**：归类错因（如计算失误、概念不清、审题错误等）。
3. **视频推荐**：**分步讲解前**，优先通过 `web_search` 搜索并推荐 1-2 个国内微课视频（B站/国家中小学智慧教育平台），或给出精确搜索关键词。
4. **分步讲解**：逐步推导，并结合 ASCII / SVG / 表格等方式进行可视化讲解。
5. **【核心断点】交互确认**：分步讲解后**必须暂停**，主动询问学生是否完全搞懂。**严禁在学生确认理解或主动提出前就输出练习题**。
6. **举一反三与归档**：学生确认理解后，生成 3-8 题分层练习（基础 -> 提高 -> 挑战），并归档错题或记录学习日志。

### Rule 3. 学生目标统一
*   **学生目标信息**（目标分数、目标专业组、选科组合等）统一在 `profile.md` 中进行 Markdown 表格管理，方便学生手动直接查看与编辑。
*   `colleges/` 目录下**严禁创建或保存 `candidate-info` 个人信息文件**，实现个人隐私与公共招生数据的物理隔离。

### Rule 4. 参考材料标记隔离
*   老师给的标准答案、范文等参考资料，存储到学科目录下时，必须在 YAML frontmatter 中显式标记 `reference_type` 字段（如 `standard_answer`/`model_essay`/`classmate_essay`），**此类文件绝对不能纳入 `_index.md` 的错题和正确率统计**。

## 🛠️ 快速排障 (Troubleshooting)

*   **图片质量低/看不清**：提示学生补充上传清晰照片，或以文字形式发送题目，不强行盲猜。
*   **学科主题不在内置列表**：先生成通用型知识讲解，并提示学生告知具体版本/章节或原题后进一步精细辅导。
*   **视频资源搜索未果**：降级为给学生提供 B站/国家中小学智慧教育平台的精确搜索关键词。
*   **薄弱点掌握度流转**：当且仅当学生在练习中**连续3次答对**该知识点，才可将 `_index.md` 中对应的状态由 `⚠️ 待巩固` 修改为 `✅ 已掌握`。
