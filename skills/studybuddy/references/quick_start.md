## 通用规则

### 练习文件格式
练习文件统一输出为 Markdown，含题目、答案、易错提醒三部分：

```markdown
# [学科] [知识点] 练习

> 学生：[姓名] | 年级：[高中X年级] | 难度：基础/提高/挑战

## 题目

### 1. (基础)
[题目内容]

### 2. (提高)
[题目内容]

...

## 答案与解析

### 1. [答案]
[解析]

...

## 易错提醒

- [易错点1]
- [易错点2]
```

文件保存到 `~/.studybuddy/subjects/<学科>/YYYY/MM/` 目录下，文件名格式为 `<YYYY-MM-DD>-<descriptive-slug>.md`（如 `2026-07-12-newtons-second-law.md`）。

### 核心教学顺序（难题破解五步法）
错题辅导严格按以下顺序，不可跳步：

```
错题识别/批改 → 错因分析 → 可视化讲解 → 视频推荐 → 举一反三练习
```

- **可视化讲解**：可输出 SVG 图形、几何示意、受力分析图等，也可标注原图
- **视频推荐**：有 `web_search` 时必须搜视频；无该工具时降级为给搜索关键词
- **举一反三**：单次不超过 8 题，三层难度梯度

### 出问题怎么办

| 问题 | 处理 |
|------|------|
| 图片看不清 | 补发清晰图，或把题目文字打出来 |
| 主题不在内置列表 | 先生成通用练习；补充教材版本/章节/原题后可细化 |
| 视频搜不到 | 给B站/国家中小学智慧教育平台/北京中小学智慧教育平台搜索关键词 |
| 信息不足（教材版本/题目缺失） | 先给用户一条可复制的补充话术，不要强行猜测 |
| 命中历史薄弱点 | 检查 `_index.md` 薄弱点表，若知识点存在且状态为 ⚠️ 待巩固，高亮提示该知识点过往错题 |
