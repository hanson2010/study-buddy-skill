# 优质解题视频资源推荐指南

## ⚠️ 重要提示（硬性要求）

**每次错题讲解后，必须执行视频推荐流程：**

1. **若环境提供联网搜索工具时**：优先锁定 **B站（第一个推荐平台）** 搜索相关知识点讲解视频（检索词加入 `bilibili` 或使用 `site:bilibili.com`），并**必须给出 1-2 个具体的视频页链接**（`https://www.bilibili.com/video/BVxxxx`，附标题、时长、推荐理由）。**禁止仅提供 B站搜索页链接来代替具体视频**——只有完全无法获取到具体视频链接（无搜索工具 / 无任何结果）时才允许走降级策略。
2. **无联网搜索工具或联网失败时（降级策略）**：提供搜索关键词，引导用户在 B站/国家中小学智慧教育平台/北京中小学智慧教育平台自行搜索

如未推荐视频或未给搜索关键词，视为讲解流程不完整。

> [!NOTE]
> **术语说明**：「联网搜索工具」指 agent 运行环境提供的实时联网检索能力（如 QoderWork、WorkBuddy、Trae Work 等平台各自提供的联网搜索工具，具体名称可能不同）。若环境未提供该能力或联网失败，一律走下方降级策略。

### 高中选片原则

- **高考导向**：优先推荐标注「高考」「一轮复习」「二轮复习」的视频，贴近考试要求
- **真题优先**：含高考真题讲解的视频优先于纯概念讲解
- **效率至上**：高中生时间紧，优先 5-20 分钟的精讲视频，超过 30 分钟需在推荐理由中说明必要性
- **难点聚焦**：针对薄弱点推荐专题突破视频，避免泛泛而谈的综合课

## 二、视频搜索关键词模板（高中）

### 通用 B站检索式（有联网搜索工具时优先使用）

直接让搜索结果命中 B站视频页，提取其中 `https://www.bilibili.com/video/BVxxxx` 形式的链接作为可点击视频链接：

```
"[知识点] 高中[学科] 讲解 site:bilibili.com"
"[知识点] 高考 一轮复习 bilibili"
"[知识点] 高中[学科] 真题 讲解 bilibili"
```

**优先参考推荐 UP主（B站必遵的选择算法）**：上方「一、推荐视频平台 → B站 → 推荐UP主」按学科列出了经过筛选的优质 UP主。对 B站，必须按以下顺序产出 1-2 个**具体视频链接**：

1. **检索**：用通用 B站检索式（可追加头部 UP主名提高命中）搜索，得到一组候选视频。
2. **优先推荐 UP主**：从候选中筛选 UP主属于「推荐UP主」列表的视频，优先挑选 1-2 个（须满足高中选片原则）。
3. **择优兜底**：若候选中**没有**来自推荐 UP主 的视频，则按高中选片原则从其余结果中择优挑选 1-2 个。
4. **提取具体链接**：对每个选定视频，提取其 B站视频页链接 `https://www.bilibili.com/video/BVxxxx`（**不是搜索页链接**），连同标题、时长、推荐理由一并给出。

> 检索式层面（可选增强）：在 query 中追加学科对应的头部 UP主名可更精准命中，例如：
> ```
> "[知识点] site:bilibili.com 赵礼显老师"   # 数学
> "[知识点] site:bilibili.com 李永乐老师官方" # 物理
> "[知识点] site:bilibili.com 化学老师李政"   # 化学
> ```

## 四、视频筛选标准

### 优质视频特征
- ✅ 讲解清晰、逻辑性强
- ✅ 有图示或动画辅助
- ✅ 时长适中（5-20分钟最佳）
- ✅ 播放量高、好评多
- ✅ 来源于认证UP主或官方机构

### 避免推荐
- ❌ 讲解模糊、跳跃大
- ❌ 广告过多
- ❌ 画质或音质太差
- ❌ 内容过时或不准确

## 七、注意事项

1. **版权意识**：只推荐公开免费的视频资源
2. **内容审核**：确保推荐视频内容正确、健康，符合高考大纲要求
3. **时效性**：定期检查视频链接是否有效，关注高考政策变化
4. **适配性**：根据学生年级和复习阶段推荐
   - **高一**：侧重概念理解和基础题型
   - **高二**：侧重知识综合和专题突破
   - **高三**：侧重高考真题、一轮/二轮复习、应试技巧
5. **多样性**：同一知识点推荐不同风格的视频，满足不同学习习惯
6. **效率意识**：高中生课业繁重，避免推荐过长或内容重复的视频
