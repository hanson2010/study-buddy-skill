# 古诗文记忆与理解 Workflow

本文件定义古诗文学习的独立流程，适用于古诗词和文言文的背诵、理解与鉴赏。

---

## 一、学习目标

| 目标 | 说明 |
|------|------|
| **背诵** | 准确背诵全文，无错字漏字 |
| **理解** | 理解字词含义、句子意思、文章主旨 |
| **鉴赏** | 分析表达技巧、意象意境、情感主旨 |
| **应用** | 能够默写、翻译、仿写、答题 |

---

## 二、学习流程

### 1. 初读感知
- **通读全文**：通读一遍，把握整体节奏和情感
- **标注难点**：标记不认识的字词、不理解的句子
- **整体印象**：初步判断文体（诗/词/文言文）、作者、创作背景

### 2. 字词解析
- **重点字词**：解释关键字词的含义（一词多义、词类活用、通假字）
- **句式分析**：分析特殊句式（判断句、被动句、倒装句、省略句）
- **词语积累**：整理重要实词、虚词、固定搭配

### 3. 句意翻译
- **逐句翻译**：准确翻译每句话的意思
- **直译意译**：区分直译和意译，说明翻译技巧
- **关键句解析**：重点解析千古名句、主旨句、过渡句

### 4. 内容赏析
- **结构分析**：分析文章结构、段落层次、过渡衔接
- **表达技巧**：分析修辞手法（比喻、拟人、夸张、对偶等）、表现手法（借景抒情、托物言志、对比衬托等）
- **意象意境**：分析意象选取、意境营造、画面感
- **情感主旨**：分析作者表达的情感、文章主旨、思想内涵

### 5. 情境联想
- **创作背景**：介绍作者生平、创作背景、时代背景
- **情境还原**：还原当时的场景、作者的心境
- **关联拓展**：关联同类作品、对比阅读、拓展阅读

### 6. 记忆巩固
- **背诵技巧**：提供记忆方法（分段背诵、关键词记忆、画面记忆）
- **默写练习**：生成默写练习题（填空、补句、全篇默写）
- **检测评估**：检查背诵和默写效果，标注错误

## 五、背诵技巧库

### 分段背诵法
将诗文按内容分段，逐段背诵，再整合全篇。

### 关键词记忆法
提取每句的关键词，通过关键词串联记忆。

### 画面记忆法
想象诗中描绘的画面，通过画面联想记忆诗句。

### 节奏记忆法
把握诗文的韵律节奏，通过朗读加深记忆。

### 理解记忆法
在理解的基础上记忆，理解越深，记忆越牢。

## 六、默写练习格式

### 填空题
```markdown
## 默写填空

1. [作者]《[标题]》中，描写[场景]的句子是：____，____。
2. [诗句前半句]，____。（[作者]《[标题]》）
```

### 补句题
```markdown
## 补全诗句

1. [诗句前半句]，________。
2. ________，[诗句后半句]。
```

### 全篇默写诗
```markdown
## 全篇默写

请默写 [作者]《[标题]》全文：

________
________
________
________
```
