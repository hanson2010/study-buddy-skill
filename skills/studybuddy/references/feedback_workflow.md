# 成绩分析反馈 Workflow

本文件定义成绩分析报告和单科卷面分析报告的处理流程，支持用户上传考试成绩分析报告或单科卷面分析报告图片，进行智能分析和个性化复习建议。

---

## 前置流程：图片处理与 OCR 识别

当用户上传成绩分析报告或单科卷面分析报告图片时，**必须**先经过 [image_workflow.md](image_workflow.md) 的图片处理与 OCR 识别流程，将图片内容转换为文字后，再进入报告分析流程。

### 流程图

```
用户上传报告图片 → [image_workflow.md] 图片预处理 → OCR识别 → 生成报告格式JSON → [feedback_workflow.md] 报告分析
```

### OCR 输入输出规范

**输入**：成绩分析报告或单科卷面分析报告图片（支持 jpg、png、pdf 等格式）

**输出**：报告格式 JSON（见 image_workflow.md 中「成绩报告格式」），必须包含以下字段：

| 字段 | 说明 | 是否必填 |
|------|------|----------|
| `content_id` | 内容唯一标识 | 是 |
| `image_path` | 图片存储路径 | 是 |
| `ocr_text` | 完整识别文本 | 是 |
| `scene_type` | 场景类型，固定为 `"成绩分析"` | 是 |
| `report_type` | 报告类型（`score_report`/`subject_report`） | 是 |
| `exam_name` | 考试名称 | 是 |
| `exam_date` | 考试日期 | 是 |

### 前置条件检查

进入报告分析流程前，必须检查以下条件：

1. **OCR 识别完成**：已获得完整的报告格式 JSON
2. **内容完整性**：考试名称、日期、成绩数据等关键信息完整
3. **图片引用记录**：`image_path` 已正确记录，方便回溯查看原始报告

### OCR 失败处理

当 OCR 识别失败或不完整时：
- 提示用户图片模糊，请求重新上传
- 提供手动输入报告内容的选项
- 识别不完整时，提示用户补充缺失内容
- **不进入报告分析流程**，直到获取完整的报告数据

## 一、报告类型

### 1. 成绩分析报告（score_report）

**适用场景**：用户上传期末考试、期中考试、模拟考试等综合成绩分析报告，包含多学科成绩汇总。

**包含内容**：
- 考试基本信息（考试名称、日期、年级、班级）
- 各科成绩（分数、满分、排名、班级平均分、年级平均分）
- 总分、总分排名、班级排名、年级排名
- 各科得分率、进步/退步情况
- 老师评语

### 2. 单科卷面分析报告（subject_report）

**适用场景**：用户上传某一学科的卷面分析报告，包含该科目的详细得分情况和错误分析。

**包含内容**：
- 学科名称、考试名称、日期
- 各题型得分情况（选择题、填空题、解答题、作文等）
- 各知识点得分情况
- 错误类型分布（概念不清、计算失误、审题错误、方法不当）
- 老师评语和改进建议

## 二、报告分析流程

### 1. 数据提取与结构化

**成绩分析报告数据提取**：
- 考试名称、日期、年级、班级
- 各科成绩：分数、满分、得分率、排名、平均分对比
- 总分、总分排名、进步/退步幅度
- 老师评语

**单科卷面分析报告数据提取**：
- 学科名称、考试名称、日期
- 各题型得分：选择题得分/满分、填空题得分/满分、解答题得分/满分
- 各知识点得分率
- 错误类型分布
- 老师评语

**未支持学科处理**：本技能仅支持六科（语文、数学、英语、物理、化学、生物）。提取成绩数据时，遇到未支持学科的成绩信息，简单跳过即可，不进行分析和记录；用户询问时，可告知不支持该学科。

### 2. 成绩综合分析

**各科表现分析**：
- 按得分率排序，识别优势科目和薄弱科目
- 对比班级平均分、年级平均分，评估相对水平
- 分析进步/退步趋势

**知识点掌握分析**：
- 识别得分率低于 60% 的知识点（薄弱点）
- 识别得分率高于 80% 的知识点（优势点）
- 分析各知识点的错误类型分布

**排名分析**：
- 当前总分排名、各科排名
- 与上次考试对比，排名变化情况
- 在班级、年级中的位置评估

### 3. 薄弱点识别与标记

**薄弱点标记规则**：
- 得分率 < 60% 的科目或知识点 → ⚠️ 待巩固
- 得分率 60%-80% 的科目或知识点 → ⚠️ 需加强
- 得分率 ≥ 80% 的科目或知识点 → ✅ 已掌握

**错误类型分类**：
- 概念不清：对知识点理解有误
- 计算失误：计算过程出错
- 审题错误：未看清题目要求或条件
- 方法不当：解题思路或方法不正确
- 时间不足：未完成题目
- 粗心大意：低级错误

### 4. 个性化复习建议

**针对薄弱科目**：
- 制定专项复习计划
- 推荐相关视频资源和练习题
- 建议复习频率和时间安排

**针对薄弱知识点**：
- 详细讲解知识点核心内容
- 提供典型例题和解题思路
- 生成举一反三练习题

**针对错误类型**：
- 概念不清：建议重新学习相关概念，结合实例理解
- 计算失误：建议加强计算练习，养成验算习惯
- 审题错误：建议圈画题目关键词，分步审题
- 方法不当：建议学习多种解题方法，对比选择最优解

**时间规划建议**：
- 根据薄弱点数量和考试时间，制定复习时间表
- 建议每天复习时间分配
- 重点突破和巩固提升的时间安排

### 5. 与历史数据对比

**纵向对比**：
- 与上次考试成绩对比，分析进步/退步情况
- 对比同一知识点的得分率变化
- 评估复习效果

**横向对比**：
- 与班级平均分、年级平均分对比
- 分析与优秀学生的差距
- 设定合理的追赶目标

每次执行成绩分析报告流程后，必须将操作概要记入 `output/YYYY/MM/YYYY-MM-DD-log.md` 文件：

**记录内容**：
- **操作类型**：成绩分析报告/单科卷面分析报告

- **核心数据**：各科分数、总分、排名、新增薄弱点数量
