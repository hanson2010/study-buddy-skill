# StudyBuddy - AI 学习助手使用指南

## 全局约定（Global Convention）

### 日期选择规则

根据原始资料中可识别的日期信息或执行本工作的当前时间，确定对应的 `YYYY/MM` 文件夹：

**日期识别优先级**：
1. **图片内容中的日期**（如"2026年4月15日期中考试"、"2026.4.10月考"）
2. **用户提供的日期信息**（如"这是上周的作业"）
3. **当前系统时间**（无法识别日期时使用）

**文件夹创建**：当目标 `YYYY/MM` 文件夹不存在时自动创建，不得武断选择之前发现的文件夹。

### 文件命名规则

生成的文件一般遵循以下命名形式：

```
YYYY-MM-DD-<slug>.md
```

**格式说明**：
- `YYYY-MM-DD`：日期格式，如 `2026-07-12`，与所在 `YYYY/MM` 文件夹的日期一致
- `<slug>`：英文小写格式，使用短横线 `-` 连接，描述文件内容
- 文件后缀固定为 `.md`

**示例**：
- `2026-07-12-derivative-applications.md` — 数学导数应用错题记录
- `2026-07-12-report-weekly.md` — 学情追踪周报
- `2026-07-12-chinese-essay-review.md` — 语文作文批改记录

**特殊文件名**：
- `YYYY-MM-DD-log.md` — 会话日志（存储在 `raw/YYYY/MM/` 目录）
- `_index.md` — 索引文件（每个目录下的快速检索入口）
- `profile.md` — 学生基础信息档案

### 原始资料存储规则

- **存储位置**：用户上传的任何资料（图片、文档等），无论使用哪个子命令（aim、learn、eval、feedback）或未明确指定子命令，都必须先存入 `raw/YYYY/MM/` 目录，不创建 assets 子文件夹
- **文件命名**：使用原始文件名存储，不计算 MD5 哈希值
- **冲突处理**：当原文件名存在时，通过添加数字后缀避免原文件被覆盖（如 `photo.jpg` → `photo_1.jpg`）
- **路径引用**：处理结果文件中必须引用正确的相对路径（`raw/YYYY/MM/<文件名>`）；如果文件名有修改，相应修改引用路径

### 子命令体系

本技能提供 5 个子命令，用于启动不同的工作流程：

| 子命令 | 功能 | 适用场景 |
|--------|------|----------|
| `aim` | 目标对齐 | 设定学习目标、上传高校资料、制定学习规划 |
| `learn` | 知识点学习与巩固 | 学习知识点、复习薄弱点、古诗文学习 |
| `eval` | 作业批改与错题归档 | 上传作业/试卷照片、请求批改错题 |
| `feedback` | 成绩反馈 | 上传成绩分析报告、单科卷面分析报告 |
| `report` | 学情追踪 | 生成学习周报/月报、分析学习趋势 |

**使用方式**：
- **命令形式**：直接输入子命令，如 `aim`、`learn`、`eval`、`feedback`、`report`
- **自然语言形式**：用自然语言描述需求，如"帮我设定目标"、"我要学习导数"、"帮我批改作业"、"分析我的成绩"、"生成学习周报"
- **图片触发**：上传图片时自动识别内容类型并匹配对应的工作流程

## 欢迎使用 StudyBuddy

StudyBuddy 是面向高中生的 AI 学习助手，专注于六科（语文、数学、英语、物理、化学、生物）的作业批改、错题辅导和练习生成。通过文件系统实现持久化记忆，让 AI 记得你的薄弱点，提供个性化学习建议。

## 快速开始

### 首次使用

```text
我是高一学生，教材版本是人教版，请帮我记录学习信息。
```

### 作业批改

```text
这张数学卷子帮我批改。请按：错题、错因、可视化讲解、推荐视频、举一反三练习。
```

### 错题辅导

```text
孩子高一，物理"牛顿第二定律"总错。请先讲明白，再出8题以内练习，最后给答案和易错提醒。
```

### 作文批改

```text
这是我写的作文《坚持》，请帮我批改并给出升格建议。

[作文内容]
```

### 古诗文学习

```text
帮我学习杜甫的《登高》，包括背诵、理解和鉴赏。
```

### 练习生成

```text
生成一份语文"阅读理解"练习，要有答案和易错提醒。
```

### 参考材料保存

```text
这是老师给的数学单元测试标准答案，请保存为参考材料，方便对照学习。
```

### 学习进度查询

```text
查询我最近一个月的数学学习情况，包括正确率、薄弱点和复习建议。
```

## 智能辅导流程

每次辅导或学习，StudyBuddy 都会按以下「智能辅导六步法」进行，并配合**交互式断点控制**进行（根据 learn 或 eval 场景进行适配，不适用时可跳过特定步骤）：

1. **诊断/识别** — eval: 识别题目并判断对错; learn: 诊断薄弱考点或学习诉求
2. **错因/盲区分析** — eval: 归类错因; learn: 分析常见盲区与易错考点
3. **视频推荐** — 在分步讲解前，先推荐 1-2 个国内平台的概念/方法类微课视频或提供搜索关键词，帮助学生快速复习背景知识
4. **分步讲解** — 采用分步文字 + 可视化表示（SVG图形/几何示意/表格等）讲解
   - **交互断点**：在第 4 步分步讲解后**必须暂停**，主动请用户确认是否搞懂、是否还有疑问。严禁在未经用户确认理解或要求前就显式输出练习题目。
5. **举一反三** — 在用户确认搞懂且做好准备后，实时生成 3-8 题梯度练习巩固知识
6. **归档/同步** — eval: 归档错题并更新学情; learn: 记录日志并更新薄弱点掌握度状态

## 核心特色

### 长期记忆
StudyBuddy 通过文件系统实现持久化记忆，一个月后依然记得你的薄弱点，提供个性化学习建议。

### 视频推荐
每次错题讲解后，都会推荐相关知识点的讲解视频，帮助你从不同角度理解问题。

### 参考材料隔离
标准答案、范文等参考材料仅作学习参考，不会影响你的学习记录和薄弱点分析。

### 分层练习
练习题分为三层难度：基础、提高、挑战，循序渐进提升能力。

## 数据存储

你的学习数据存储在以下位置：

| 文件 | 说明 |
|------|------|
| `~/.studybuddy/profile.md` | 你的基础信息（姓名、年级、教材版本、学习目标） |
| `~/.studybuddy/_index.md` | 学习状态总览（考试成绩、全局薄弱点） |
| `~/.studybuddy/colleges/` | 高校资料（目标院校专业组、投档数据、分数分布等外部数据） |
| `~/.studybuddy/subjects/<学科>/_index.md` | 各学科学习状态（高频主题、薄弱点表） |
| `~/.studybuddy/subjects/<学科>/YYYY/MM/*.md` | 练习和错题记录 |

> 你可以通过设置环境变量 `STUDYBUDDY_DATA_DIR` 指定数据存储路径。

## 使用技巧

1. **拍照上传**：可以直接拍照上传作业或试卷，AI 会自动识别题目
2. **标注重点**：批改时可以指定重点分析的题目类型
3. **补充信息**：如果题目不完整或图片模糊，AI 会主动询问补充信息
4. **复习建议**：定期查询学习进度，根据薄弱点进行针对性复习
5. **参考学习**：保存标准答案和优秀范文，方便对照学习

## 常见问题

### Q: StudyBuddy 支持哪些教材版本？
A: 支持人教版、苏教版、北师大版等主流教材版本。首次使用时请告知教材版本。

### Q: 视频推荐来自哪些平台？
A: 主要来自 B站、国家中小学智慧教育平台、北京中小学智慧教育平台等优质教育平台。

### Q: 我的学习数据安全吗？
A: 你的学习数据存储在本地电脑，不会上传到云端，确保数据安全。

### Q: 如何重置学习数据？
A: 删除 `~/.studybuddy/` 目录即可重置所有学习数据。
